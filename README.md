# Utopia Scoreboard

A cross-platform scoreboard for Mahjong, Card Game or Table Game.  
跨平台的计分器，可用于麻将、扑克、桌游等。  

## 开发初衷
过年天天打麻将斗地主。新时代新青年咱们不赌钱，但是总得计分才知道谁输谁赢谁运气好谁技术差，所以开发一个计分板来简单记录一下。
